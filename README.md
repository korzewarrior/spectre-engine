# Spectre Engine — private landing page

This is a lightweight, dependency-free landing page for **Spectre Engine**.

## Run locally
Just open `index.html` in a browser.

## Edit the copy
Open `index.html` and tweak text in the hero + sections.

## Set your real contact email
Open the page → click **Request early access** → hit **edit** and set your email once.
(Or hardcode it in `index.html` by replacing `hello@spectre.engine`.)

## Files
- `index.html` — structure + copy
- `css/styles.css` — black/white sharp styling
- `js/app.js` — background canvas, command palette (Ctrl+K), theme toggle, modal
- `assets/favicon.svg` — simple sharp icon

## Notes
- Includes `meta robots=noindex,nofollow` by default (good for private staging).
- No analytics / tracking / external CDNs.
